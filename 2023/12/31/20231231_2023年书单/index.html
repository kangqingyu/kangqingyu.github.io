2023年书单

2023年12月31日

2023年阅读41本书，阅读书籍的类型增加了，不再像2022年主要是看学习效率类的。

2023还看了文学名著、管理学、心理学。本文推荐我认为评分在7分以上的书。

《红楼梦》

(清) 曹雪芹 著， (清) 无名氏 续; (清) 程伟元，高鹗 整理 / 人民文学出版社
一本小说多个主题，反应了复杂的人性，丰富、现实的生活。1封建时代的百科全书。2宝黛爱情故事。3凤姐能力再强，也无法与封建礼教抗衡。有时候看似赢得了表面，风光无限，但实际输了人生。4贵族从兴盛到衰落的历史，回头来落了片白茫茫大地真干净

《水浒传》

施耐庵 人民文学出版社
表面看是一群英雄好汉的故事。
但仔细看似乎又是一本宣扬暴力，崇尚暴力的书，是一本儿不讲道理，不讲逻辑的书。
虽然有的好汉也讲道理，但他讲的道理往往经不起逻辑的推敲。
虽然有的好汉也讲逻辑，但他讲的逻辑往往又是强盗的逻辑。
虽然有的好汉也行侠侠义，但更多的好汉却是作奸犯科之辈。
虽然有的好汉是被逼无奈，但有的好汉却是主动为非作歹，虽然也讲义，但又少了正义。
虽然讲了忠，却又尽是伪忠，唯有暴力贯穿始终。
好汉们解决问题的方式，大多是实施暴力，而对方一定要支付血的代价。谁狠谁就牛，谁就受人尊敬。作者似乎不是在教你做人做事，而是在教你如何使用暴力，或者熏陶欣赏使用暴力。

再仔细看人物的语言和行为分析，又会发现很多人物的阴暗面。
梁山排名的现象如今也非常真实：座次排名和岗位职位差别挺大。排名偏下，岗位职务可能更好。有的好汉在排名上无法靠前，但在岗位上却占据非常好的要职，这要完全取决于与宋江的私人关系。

人物虽然都不完美，但这不就是真实的世界吗？总之，这是反应老百姓阶层的人生思想与人性的好故事。

《未来简史》

[以色列] 尤瓦尔·赫拉利 / 林俊宏 / 中信出版社
过去几千年来想象中秩序，具有重大意义；而在科技发达之后就要 放弃意义、换取力量。不同时代有不同的活法，有收获，就得有代价。感受也是一种算法，或者并不被人所知罢了，正如大家都以为自己没有信仰，其实无论是古代对秩序，还是现在对于金钱的追求，这就是一种自己不知道的信仰。

《人类简史》

尤瓦尔·赫拉利 / 林俊宏 / 中信出版社
从人类祖先狩猎时代，讲到了农业社会，以及现在科技的发展。从更广的角度讲了人类的发展史，指明了宏观了方向。基因的传承不在于个体的快乐与感受，而这正是个人非常在意的。

《翦商》

李硕 / 广西师范大学出版社
历史没有真相，本书不仅讲解了石器时代到周朝的历史。作者还通过一步步展现在世人面前的考古发现，用独特的思考视角，揭示出被埋藏两千多年的真相。或许在美好生活之下，真相已不重要

《思考，快与慢》

丹尼尔·卡尼曼 / 胡晓姣 / 中信出版社
作者从大脑快慢两种思考系统讲起，先介绍常见的偏见，再进一步讲解决策中的过度自信问题与常见错误，让我们能够在各种决策中，能够清晰认识自己，平衡风险与收获，做好更好的决策。改变世界的第一步是认识自己。

《影响力》

罗伯特·西奥迪尼 / 闾佳 / 北京联合出版公司
从心理学底层原理，利用大脑的思维方式。介绍了随着文明的进步越来越多的8种无意识的自动行为。一方面可以利用这些来提高自己的影响力，另一方面可以预防别人设下的自动化陷阱。另外我还想到了一种应用：用来将学习这一类困难的行动，转化为零成本的自动化的无意识行动。

《贪婪的多巴胺》

(美) 丹尼尔·利伯曼 / 郑李垚 / 中信出版社
我想再吃一块甜品，但是吃完之后我并不快乐。我想买辆豪车，但是贷款买了之后我并不快乐。
为什么放纵自己的欲望并不会带来快乐？为什么我苦苦追求的东西，得到了之后并不快乐？
人体内的两种物质：多巴胺和当下神经递质。
1多巴胺产生对未来资源的渴望，促进自己付出努力。
2当下神经递质在拥有之后产生相应的满足的感受。
多巴胺产生的渴望，并不在乎用什么手段，即使欺诈与武力，也是可以使用的工具。虽然多巴胺系统进化出来的目的，是为了人类生存和繁殖。但那是几万年前的产物；在21世纪物质丰富的年代，不停的追求过盛的甜食已不再适用，因为我们已不再处于饥饿的边缘。
正确使用多巴胺，可以像信贷一样扩大生产再建设；用的不好就会陷入消费主义无尽的欲望深渊。

放纵自己的欲望并不一定会带来快乐，因为你想要的不一定是你喜欢的。为我们要克服对“更多”欲望的痴迷，克服多巴胺刺激的诱惑；学会享受我们拥有的东西。将多巴胺和当下神经递质结合，达到和谐的程度。
运动、画画、创作音乐、制作飞机模型、烹饪、园艺都可以将智力刺激与体力活动结合起来，这些活动不仅能产生快乐和满足，财富和知识，还能产生丰富的感官体验和智慧，这可以让我们走上一条更平衡的人生道路。

《冥想》

斯瓦米·拉玛 (Swami Rama) / 刘海凝 / 天津人民出版社
看了其他几本冥想入门书，都是云里雾里的。这本从入门到进阶（理论）讲的都非常清楚。冥想入门必读。

《改变的历程》

乔·迪斯派尼兹（Joe Dispenza） / 凌春秀 / 人民邮电出版社
看了几本冥想的书，这才是一本真正知行合一的书。不但有科学的原理，还有具体的实践方式。 其他的书，有些是是从很丰富经验的角度视角写，让初学者感到云里雾里，而有些书只则简单说了动作，但不知道为什么要这样做。

《阿里巴巴管理三板斧》

王建和 / 机械工业出版社
一 作为一家大型互联网公司，人员的快速增加对组织最大的挑战是缺乏管理者。通过很多销售的例子， 讲了阿里的管理。
1 阿里如何培养大量优秀的管理者？管理者不同于技术和资本，不能依赖进口，管理者只能培养。中基层管理者不要空降，而要自己培养，没有其他捷径。今年缺少人才，是3年前决定的。（有些公司则认为是筛选，选出有天赋的管理者，不过其前提和阿里是一样的：需要招到大量优秀的人才）
2 管理是一种实践，其本质不在于知，而在于行，其验证不在于逻辑，而在于成果，其唯一的权威就是成就。
3 初级管理者要理解中高层管理模糊的旨意，用清晰的方式向下级转达。
4 管理的本质就是激发一个人的善意和良知；要不断地激发人的责任、价值、成就，最终实现自我管理，即谷歌提出的赋能。通过同理心，从对方的利益角度思考问题，以达到激发其工作动力的目的。
二 如何保持时刻有大量优秀的人才，关键是人才的招聘与维持。通过定期清理小白兔，避免死海效应。最终祸及整个公司。小公司的成败在于你聘请什么样的人，大公司的成败在于你开除什么样的人。阿里开人真是一种深入骨髓的文化。
三 文化价值观：用公司的钱，让员工之间互相关怀。通过各种细节来关怀人、留住人。满足人的各种层次的需求。

《重新定义公司》

埃里克·施密特 / 靳婷婷 / 中信出版社
讲了谷歌成为顶级公司的几点原则。当谷歌从创业团队成长为巨大的公司时，依然保持了这些文化与价值观。可见一个团队创始人拥有的价值观是多么重要。 1.招聘最优秀的成长型思维的创意精英。通过权利的拆分，部门内只有否定权、而没有招聘权，避免因个人利益而造成公司的损失。2.组织赋能，而非管理或激励。激励偏向事成之后的利益分享，需要延迟满足。而赋能强调激起创意人的兴趣与动力，给予挑战。发自内心的志趣才能激发持续的创造。3.即使决策已是板上钉钉，也要提供发表意见和投票表决的权利。讨论，管理者最后再发言，提供大家都发言的环境。达成对公司最有利的决策的共识。晓之以理、动之以情的“奥普拉温弗瑞法则”。4.管理者通过各种手段寻找最优秀的人，打造任人唯贤的环境，避免“河马”的影响。并要始终坚持这种原则。

《小米创业思考》

雷军 / 中信出版社
推广介绍了小米的各种产品。小米的方法论与价值观，为人民利益考虑，以及降低各行业成本方案的探索。雷布思是一位有理想的企业家

《小强升职记》（升级版）

邹小强 / 电子工业出版社
用更详细的例子介绍了GTD，相比《搞定》更容易入门。适合GTD新人快速理解，不会被《搞定》复杂抽象的理论吓退。

《升职，凭什么是你》

[美] 尼亚姆·奥基夫 / 彭剑 / 机械工业出版社
升职必备7要素，包含了不少普通人不知道的“小秘密”。

《非暴力沟通》

马歇尔·卢森堡 / 刘轶 / 华夏出版社
作者讲了非常新颖的沟通方式。不带评论的观察，即没有主观的判断。注意自己的感受需求，沟通时注意同理对方，感受对方的需求。原来生气的原因是面对不喜欢的言语或行动，选择了指责对方。此时应该同理他人。不论是在生活中的沟通，还是开会时沟通，这些技巧都非常实用。

《俞军产品方法论》

俞军 / 中信出版集团
作为一名低调的产品届教父，俞老师从产品经理必备的素质，讲到如何根据不同的企业有不同的成长。不仅有不同产品不同阶段的特点介绍，还有选择offer时从导师，业务，平台几方面的权衡与选择。产品的方方面面都有详细讲解。各种经济学、心理学与产品融会惯通。可见俞老师读书之多、理解之深刻。

《我在北京送快递》

胡安焉 / 湖南文艺出版社
一位经验丰富打工人的成长经历，描述了真实的历程以及感受。让我们看到不同行业 ，不同的现状。有人的地方，就有江湖。珍惜当下美好的工作机会。本书的缺点是后部部分结构松散，不过对于会高效阅读的读书人来说，影响不大。

《把时间当作朋友》

（第3版）李笑来 / 电子工业出版社
说实话的勇士。虽然个别观点不认同，但是大部分都是内心，非常真诚

《心的力量》

刘嘉森 / 江苏凤凰文艺出版社
衡水高考状元亲身的讲述。确实，阅读和性格非常重要。在不确定性中对抗挫折的能力在高段位博弈过程很重要。

《乌合之众》

(法) 古斯塔夫·勒庞 / 陆泉枝 / 上海译文出版社
作者独特的观点从另一个角度说明了历史没有真相，历史书可能只是群众心中想象的英雄的故事，而并非真实的历史故事。是对有误观察的离奇记述、再掺上事后给出的解释。 2.人类阶级金字塔，越底层越受信仰的影响、其思考能力也越弱。 而顶级精英并非靠严谨的逻辑、而是靠声望来传播。即使很注重逻辑性，但是理性相比感性来说，力量太弱了。因追逐梦想而从野蛮步入文明，当梦想丧失力量便走向衰亡，这正是一个民族的生命历程。 3.历史是无限循环的，人类从中唯一吸收的教训就是“人类不会从历史中吸取教训”。

《动物农场》

[英]乔治·奥威尔 / 荣如德 / 上海译文出版社
历史没有真相，一直在循环。除非你是人，猪或者狗，否则没必要陷入意识形态的争论中，放松放活好当下。 多读经典，多读历史，太阳底下没有新鲜事，你遇到的任何问题，前人都遇到过。成长型思维，一直学习新适应，了解万事万物运行的底层原理和本质。有独立思考的能力就不容易被忽悠。 无论怎样，还是会有不如意的地方。世界上只有一种真正的英雄主义，那就是认清生活真相依旧热爱生活。

《人人都能用英语》

李笑来
讲了很多理论，确实能在一定程度上提升效率。但其实关键还是要使用，要实践中使用，通过不断反馈、获得成就来达到熟能生巧的程度。

《韭菜的自我修养》

李笑来 / 江苏凤凰文艺出版社
介绍投资的一些基本常识，其实关键还是要战胜自己的贪欲，有长远的眼光。能够承受痛苦并且反思，这样才能有长远的成长

《谁动了我的奶酪?》

[美] 斯宾塞·约翰逊 / 魏平 / 中信出版社
拥抱变化，热爱变化，时刻为变化做准备。变化无处不在，迅速适应、而非被动迎接变化可以使我们占得先机。关键是观念，要相信观念能带来长远的收益

《肺话》

杨跃 人民卫生出版社
一本医学科普好书。介绍了雾霾情况下口罩、洗鼻、空气净化器等方面的防护，要养成良好的生活习惯，避免看不见的大颗粒污染物。介绍了多种饮食方式、运动方式、护肤品的使用，避免单一方向小错误积累导致的巨大危害。从医生角度表达了医患关系常见的问题。也最后推荐了对肺较好的食物。

《爱上收纳：井井有条又热气腾腾的家》

蚂小蚁 / 化学工业出版社
前部分讲述自己家里故事内容太多了。最后一部分收纳的基础知识不错。以前物品随意堆放。现在开始用收纳的知识整理一下自己的物品，这和电子产品文件夹的分类与整理原理比较类似。

《认知觉醒》

周岭 / 人民邮电出版社
各种成长方法论的汇总。虽然后半部分结构有些松散，每个理论部分较独立，没有很好得结合在一起，实际例子也很少。但相比大部分只关注某一方面的成长类书来说，涉及面更广。对不同的方法论有了较多的联结。

《认知驱动》

周岭 / 人民邮电出版社
讲了从内部驱动行动的各种方法论。拥有长期主义的思想，才能走得更远。和前一本有一部分重合的内容，不过从驱动、动力的角度来看，总结得还比较全面。

《芯片战争》

克里斯·米勒 / 蔡树军 / 浙江人民出版社
不同时代需要不同的人才，而在芯片时代，需要的是有高度创新的人才。这要求良好的环境，培养大批的人才。这不是一个短期的投入，而是一个长期的投入，毕竟这是未来重要的战争。
